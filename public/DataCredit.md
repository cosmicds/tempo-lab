# Data sources

- `states.geojson` - U.S. State boundaries from [eric.clst.org](https://eric.clst.org/tech/usgeojson/). We use the `500k` GeoJSON file. This sacrifices detail for performance. Newer US state boundaries are available from the [US Census Bureau](https://www.census.gov/geographies/mapping-files/time-series/geo/cartographic-boundary.html#titlecore-d6c03f5902).

- `coastlines.geojson` - 1:50m (medium resolution) Coastline data v4.0.0 from [Natural Earth](https://www.naturalearthdata.com/downloads/50m-physical-vectors/)
